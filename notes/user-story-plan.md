### 1.2 用户故事体系的商业价值

```
用户痛点：民宿老板不信广告，只信同行说的话

用户故事作用：
1. 信任代理 → 同行真实经历比广告强100倍
2. 场景共鸣 → "这不就是我的情况吗" → 转化加速
3. 搜索引擎 → 长尾词"民宿装了防飞房系统后悔了吗" → 天然SEO
4. 差异化护城河 → 竞品无法快速复制（需要真实用户积累）
```

## 三、用户故事体系设计方案

### 3.1 板块架构（三层结构）

```
┌─────────────────────────────────────────────────┐
│  L1：首页证言轮播区（已建成框架）                   │
│  入口：首页 → 3-5条精选证言滚动展示                │
│  作用：快速建立第一印象信任                        │
└─────────────────────────────────────────────────┘
          ↓ 点击"查看更多故事"
┌─────────────────────────────────────────────────┐
│  L2：案例故事列表页（cases.html 新建）            │
│  入口：导航"案例" → 瀑布流/卡片列表                │
│  内容：封面图 + 民宿名 + 核心数据 + 标签            │
│  作用：让用户找到与自己情况相似的案例              │
└─────────────────────────────────────────────────┘
          ↓ 点击具体案例
┌─────────────────────────────────────────────────┐
│  L3：案例详情页（case-{id}.html 或动态加载）       │
│  入口：从列表页进入                               │
│  内容：完整用户故事（民宿背景→痛点→使用过程→结果） │
│  作用：深度说服，解答"我装了会怎样"的核心疑问      │
└─────────────────────────────────────────────────┘
```

### 3.2 内容模板：单条用户故事标准结构

```markdown
## 【故事标题】：让老板娘睡个好觉的民宿

### 基本信息卡
- 民宿名称：莫干山 XX 民宿
- 规模：12间客房
- 地区：浙江德清
- 使用时长：8个月
- 节能效果：月度电费下降 31%

### 故事正文

**【民宿背景】**（建立共鸣）
"我们2019年开业，前几年生意还行。2023年开始，明显感觉客人少了，
但电费账单却一年比一年高。老板娘天天念叨，说再这样下去要亏本。"

**【核心痛点】**（戳中同类用户）
- 不知道客人走了是去了别家还是压根没来
- 退房不通知、续住不确认，每月"飞房"损失超2000元
- 电费刺客：待机功耗占月度电费35%

**【为什么选领锁】**（解答决策疑虑）
"当时加了十几个品牌的微信，对比了三个月。选领锁的原因：
1. 不需要换锁，酒店公寓也能用
2. 小度音箱联动，说句话就能查房间状态
3. 销售老周是真正的民宿老板，不是纯销售"

**【使用过程】**（消除"装起来麻烦吗"的顾虑）
"装机大概花了2小时，不用改线路。第一个月其实还在观望，
没觉得有多大变化。第二个月开始看数据，发现有个房间连续3天
凌晨2点门锁有动作，但当天没人入住...原来是阿姨忘锁门了。"

**【真实数据】**（硬核说服）
- 飞房率：从 8.3% → 1.2%
- 月度电费：平均节省 680 元
- 投诉率：住客投诉"房间莫名有人进入"降至0
- 老板娘评价："终于不用半夜爬起来查房了"

**【给同类民宿老板的一句话】**
"如果你也在为民宿的能耗和安全管理头疼，别犹豫，先寄样来试试。
不装也没损失，装了真的不一样。"
```

## 四、数据采集策略（如何获取真实故事）

### 4.1 优先级排序

```
第一阶段：存量用户转化（难度：低）
└── 从现有成交用户中筛选愿分享故事的案例
    激励：免费续期1个月 + 官网实名展示

第二阶段：新签用户引导（难度：中）
└── 签合同时邀请用户填写"使用体验问卷"
    6个月后自动触发回访，邀请撰写故事

第三阶段：KOC培育（难度：高，长期）
└── 培养"领锁民宿体验官"身份
    定期组织线下聚会，沉淀故事素材
```

### 4.2 用户筛选标准

**优先征集**：
- ✅ 有具体数据（电费下降X%、飞房减少X次）
- ✅ 有前后对比（装之前vs装之后）
- ✅ 有情感共鸣点（老板娘说、老板原声）
- ✅ 经营规模中等（10-30间）的大众案例

**暂不征集**：
- ❌ 纯功能描述，无情感
- ❌ 数据模糊（"电费省了不少"不如"每月省680元"）
- ❌ 刚装1个月以内（数据不稳定）
