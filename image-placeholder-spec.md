## 三、图片上传规范

### 格式优先级
1. **WebP**（推荐，体积小，质量高）
2. **PNG**（透明背景、Logo、图标）
3. **JPG**（照片、场景图）

### 尺寸规范
- **最小边不得低于展示尺寸**
- **比例必须严格匹配**，不得变形
- **所有数字Banner图，建议高度≥320px**

### 命名规范
```
领锁-用途-位置.jpg/png/webp
示例：
- 领锁-案例-酒店前台-800x600.webp
- 领锁-产品-静音设备-1600x1200.webp
- 领锁-期刊封面-特斯拉-1200x675.webp
```
