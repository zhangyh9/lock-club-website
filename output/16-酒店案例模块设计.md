```
┌─────────────────────────────────────┐
│  NAVBAR (全站统一)                  │
├─────────────────────────────────────┤
│                                     │
│  标题区：酒店落地案例实录            │
│  简介：真实客户使用效果反馈          │
│                                     │
├─────────────────────────────────────┤
│                                     │

```

## 四、给领锁笔杆的文案要求

1. 每个案例遵循：**客户背景 → 需求痛点 → 我们的解决方案 → 落地效果** 结构
2. 用数据说话，比如改造后电费下降多少，安装时间节省多少
3. 中文英文对应，保持双语一致

---

## 五、给领锁码农的开发要求

1. 卡片网格布局，响应式适配
2. 保持和全站一致的卡片样式、悬停效果
3. 点击卡片可以看完整详情（可以直接在页面展开）
4. SEO友好，每个案例有独立标题
